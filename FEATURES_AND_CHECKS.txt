                              CORE FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. SINGLE FILE VALIDATION
   - Upload individual PDF or DOCX files
   - Real-time processing with progress indicator
   - Instant scoring and detailed results
   - Page-by-page analysis view

2. BATCH PROCESSING
   - Upload multiple files at once (drag & drop)
   - Concurrent processing with progress tracking
   - Aggregate statistics (passed/warnings/failed)
   - CSV export of all results

3. PDF PREVIEW
   - Built-in PDF viewer with zoom controls (50%-200%)
   - Page navigation with quick buttons
   - Page-by-page status indicators
   - Text content preview

4. PAGE ANALYSIS VIEW
   - Detailed justification for each page
   - Pattern matching display (matched/missing)
   - Fix suggestions for failed pages
   - Layout issues accordion

5. DARK/LIGHT THEME
   - System theme detection
   - Manual theme toggle
   - Persistent theme preference

The validator provides specific fix suggestions for:
  • Cover Page - Title formatting, student info, RRN placement
  • Bonafide Certificate - HOD signature, date, certification text
  • Internship Certificate - Company certificate requirements
  • Viva Voce - Examiner placeholders, date fields
  • Acknowledgement - Required mentions (VC, Dean, HOD, Guide)
  • Table of Contents - Chapter listing, page numbers
  • Abstract - Word count, content guidelines
  • All Chapters - Sequential numbering, required sections
